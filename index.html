Intelligent Wall Clock

This application is the controller for the Intelligent Clock™. Using it you can remotely control the device, set its time, alarms and notifications.

To set up a new alarm, simply click on the menu icon in the upper left corner of the main screen and select New Alarm. To change the time of the Intelligent Clock™ also go to the menu icon and choose settings.

If you have any questions or doubts simply contact us at somerandomemail@mail.com
